xTrader
=======

a win32 project based on CTP Api(Shanghai Futures Information Technology)

◆本软件特色:	

	MFC/C++开发,基于CTP平台接口,可交易国内四大期货交易所品种.
	加入股票接口,同样可交易沪深股市.大致特色如下:
	
	1.精简界面,热键下单,支持银期转账,以及查询历史结算单.
	
	2.NTP自动15分钟网络校时.
	
	3.行情,交易登陆后可一直保持在线,有重连机制.
	
	4.超出单笔下单限制可自动批量完成.
	
	5.所有列表均可导出为csv文件.
	
	6.慢慢发掘吧,我做了相当多细节优化.
